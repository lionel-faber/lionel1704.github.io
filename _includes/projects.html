Code For Cognizant

An Android Application to host a technical event / hackathon. Built as part of the activities of the Cognizant Student Club and will be submitted to Cognizant as a proposed application to be used for the National event.
Features include authentication using email or Social Networks (Facebook, Google), Realtime Databases and Cloud Messaging.

A web application built using Django is developed to securely access the Firebase Realtime Database to read queries and send responses to participants.




S.M.I.L.E.

Simple Mobile Instant Learning Environment

An online portal to give information and provide updates regarding class activities and academic progress. For the students, by the students.
The application works with a Django Web Application as a server that can recieve requests(GET and POST) and send JSON responses. The application handles these JSON responses to parse data into the application. Google's Firebase Cloud Messaging is used for the sending of notifications.




Adios Amigos



A Web Application built using Python on the Django Framework. It allowed the juniors to send farewell messages to their seniors. Seniors could login and read the farewell messages written exclusively for them.
