CodeWars

A very simple server - client appplication for the preliminary round of a quiz event.
Coded using basic Java, Javascript, JSTL, HTML and CSS; this application fetches a set of questions from a database randomises and displays them.
A timer is included that automatically hits the submit button after the specified time. The submit button, calculates the total marks and inserts the result along with the requested details again in the database.


Automatic Traffic Light Control System

This system aims to automate the control of traffic lights at signals according to traffic density and to capture details of speeding vehicles even in the absence of police personnel. It's a work in progress that will use Open Source hardware such as Arduino UNO
