CodeWars

A very simple server - client appplication for the preliminary round of a quiz event.
Built using basic Java, Javascript, JSTL, HTML and CSS;  this application fetches a set of questions from a database randomises and displays them.
A timer is included that automatically hits the submit button after the specified time. The submit button, calculates the total marks and inserts the result along with the requested details again in the database.




Smile Website and Mobile Application

Simple Mobile Instant Learning Environment

An online portal to give information and provide updates regarding class activities and academic progress. For the students, by the students.
The Android Applicaton provides easy access to the Smile portal with a user friendly interface both online and offline. It uses Google's firebase at the server side to store data and to send notifications via Cloud Messaging.
